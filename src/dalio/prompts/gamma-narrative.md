# Gamma 叙事大纲生成提示（Joshua 第一人称版）

## 核心设定

**叙事人称**：第一人称，主角 = **Joshua**（账户持有者本人）
**调性**：沉着 + 有温度 + 自省。像 Joshua 在年终翻自己账本，跟一个老朋友说"今年这账本告诉了我什么"。
**视觉**：每张卡尽可能让 Joshua 和家人**作为简笔画角色出现在画面上**，而不只是抽象的数字和图表。

## 任务

根据提供的财务分析报告（`analysis.md`）+ 角色档案（`personas.ts`），生成 Ray Dalio《How the Economic Machine Works》风格的中文叙事大纲——但**不是冷冰冰的"机器"叙事**，而是 **Joshua 自己讲述「我家这一年」的故事**。

## 输出规格

**必须恰好 8 张卡片**，每张卡片包含 4 个字段：

```json
{
  "title": "简洁标题（≤12 字）",
  "key_points": ["核心观点 1", "核心观点 2", "核心观点 3"],
  "visual_concept": "视觉概念描述，必须明确指出哪些角色（Joshua/NaNa/燕燕/龙/皆柏贸易）出现在画面上",
  "narration": "Joshua 第一人称旁白（100-150 字），口吻温和但有洞察"
}
```

## 调性约束

- **第一人称代词**：用"我"、"我家"、"我太太"、"NaNa"、"燕燕"、"龙"、"我们"——**禁用**第三人称的"该用户"、"账户持有者"
- **禁口语词**：不用"哎"、"哇"、"超棒"、"震撼"、"令人惊讶"、"非常"、"特别"——但**可以用**"看似 X 实则 Y" / "我才发现" / "原来" / "回头看"
- **保留 Dalio 反转钩子**：至少 3 张卡的 narration 包含"看似 X，实则 Y"或类似反转结构
- **温度感来自细节**：用具体场景（"十二月那几天给 NaNa 转的钱"）替代抽象（"大额转账"）
- **旁白节奏**：150-160 WPM，每张卡旁白朗读时长约 45-60 秒
- **Joshua 的声音**：自省但不感伤，分析但不冰冷，像跟自己对话

## visual_concept 写法（必须包含角色）

**反例**（不可用，因为没角色）：
> "柱状图显示月度支出，12 月柱形高亮"

**正例**（合规）：
> "Joshua 站在画面左侧，指向右侧的 12 月柱子；柱子用红色高亮，旁边浮现 NaNa、燕燕的小头像"

可用 Remotion 组件：
- `<StickFigure character="joshua|wife|family-female|family-male|business" pose="standing|walking|waving|pointing" label="..." />` — 简笔画人物
- `<DrawPath>` / `<DrawArrow>` — 线条/箭头
- `<GrowBar>` / `<CountUp>` — 数据图表
- `<SnapIcon>` / `<ConceptHighlight>` — 强调
- `<CameraPan>` — 镜头推拉
- `<PaperBackground>` — 米色纸背景（默认在每卡）
